# NeoForge 1.21.1 迁移计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 shanmod 从 Forge 1.20.1 迁移到 NeoForge 1.21.1

**Architecture:** NeoForge 是 Forge 的继任者，使用不同的 Gradle 插件 (ModDevGradle) 和版本命名方案。主要变更在构建配置，Java API 保持向后兼容。

**Tech Stack:** NeoForge ModDevGradle 1.0.11, NeoForge 21.1.x, Java 21, Minecraft 1.21.1

---

## 关键差异总结

| 项目 | Forge 1.20.1 | NeoForge 1.21.1 |
|------|--------------|-----------------|
| Java 版本 | JDK 17 | **JDK 21** |
| Gradle 插件 | `net.minecraftforge.gradle` | `net.neoforged.moddev` |
| 版本属性 | `forge_version` | `neo_version` |
| 配置文件 | `mods.toml` | `neoforge.mods.toml` |
| MDK 目录 | `META-INF/mods.toml` | `META-INF/neoforge.mods.toml` |
| 依赖名称 | `forge` | `neoforge` |

---

## 文件变更清单

### 需要修改的文件
- `gradle.properties` - 更新 MC/NeoForge 版本配置

### 可能需要修改的文件 (API 兼容性检查)
- `submodules/shanmod/src/main/java/com/github/maoshan/shanmod/event/ModEvents.java` - 检查 `LivingKnockBackEvent` 是否存在

## Task 3: 更新 submodules/shanmod/build.gradle

**Files:**
- Modify: `submodules/shanmod/build.gradle`

```gradle
plugins {
    id 'java-library'
    id 'eclipse'
    id 'idea'
    id 'maven-publish'
    id 'net.neoforged.moddev' version '1.0.11'
}

// Load shared properties from root gradle.properties
def rootProps = new Properties()
file('../../gradle.properties').withInputStream { rootProps.load(it) }

// Define as local variables for use in closures
def mod_version = rootProps.getProperty('mod_version')
def mod_group_id = rootProps.getProperty('mod_group_id')
def mod_license = rootProps.getProperty('mod_license')
def mod_authors = rootProps.getProperty('mod_authors')
def minecraft_version = rootProps.getProperty('minecraft_version')
def minecraft_version_range = rootProps.getProperty('minecraft_version_range')
def neo_version = rootProps.getProperty('neo_version')
def neo_version_range = rootProps.getProperty('neo_version_range')
def mapping_channel = rootProps.getProperty('mapping_channel')
def mapping_version = rootProps.getProperty('mapping_version')
def mod_description = 'A Minecraft NeoForge mod developed by maoshan.'

// Also set as project properties for other uses
project.ext.mod_version = mod_version
project.ext.mod_group_id = mod_group_id
project.ext.mod_license = mod_license
project.ext.mod_authors = mod_authors
project.ext.minecraft_version = minecraft_version
project.ext.minecraft_version_range = minecraft_version_range
project.ext.neo_version = neo_version
project.ext.neo_version_range = neo_version_range
project.ext.mapping_channel = mapping_channel
project.ext.mapping_version = mapping_version
project.ext.mod_description = mod_description

version = mod_version
group = "${mod_group_id}.${project.name}"

base {
    archivesName = project.name
}

// NeoForge ModDevGradle 配置
neoForge {
    version = neo_version

    runs {
        configureEach {
            workingDirectory project.file('run')
            property 'forge.logging.markers', 'REGISTRIES'
            property 'forge.logging.console.level', 'debug'
        }

        client {
            client()
        }

        server {
            server()
            args '--nogui'
        }

        gameTestServer {
            gameTestServer()
        }

        data {
            data()
            args '--mod', project.name, '--all', '--output', file('src/generated/resources/'), '--existing', file('src/main/resources/')
        }
    }

    mods {
        "${project.name}" {
            sourceSet sourceSets.main
        }
    }
}

// Include resources generated by data generators.
sourceSets.main.resources { srcDir 'src/generated/resources' }

repositories {
    // NeoForge Maven 自动添加
}

dependencies {
    neoForgeLibrary "net.neoforged:neoforge:${minecraft_version}-${neo_version}"
}

// Parchment 配置 (可选但推荐)
neoForge {
    parchment {
        minecraftVersion = minecraft_version
        mappingsVersion = mapping_version
    }
}

tasks.named('processResources', ProcessResources).configure {
    var replaceProperties = [
            minecraft_version: minecraft_version, minecraft_version_range: minecraft_version_range,
            neo_version: neo_version, neo_version_range: neo_version_range,
            mod_id: project.name, mod_name: project.name, mod_license: mod_license, mod_version: mod_version,
            mod_authors: mod_authors, mod_description: mod_description,
    ]

    filesMatching(['META-INF/neoforge.mods.toml', 'pack.mcmeta']) {
        expand replaceProperties + [project: project]
    }
}

tasks.named('jar', Jar).configure {
    manifest {
        attributes([
                'Specification-Title'     : project.name,
                'Specification-Vendor'    : mod_authors,
                'Specification-Version'   : '1',
                'Implementation-Title'   : project.name,
                'Implementation-Version'  : project.jar.archiveVersion,
                'Implementation-Vendor'   : mod_authors,
                'Implementation-Timestamp': new Date().format("yyyy-MM-dd'T'HH:mm:ssZ")
        ])
    }

    finalizedBy 'reobfJar'
}

publishing {
    publications {
        register('mavenJava', MavenPublication) {
            artifact jar
        }
    }
    repositories {
        maven {
            url "file://${project.projectDir}/mcmodsrepo"
        }
    }
}

tasks.withType(JavaCompile).configureEach {
    options.encoding = 'UTF-8'
}
```

- [ ] **Step 1: 重写 submodules/shanmod/build.gradle**
  - 将 `net.minecraftforge.gradle` 插件改为 `net.neoforged.moddev` version `1.0.11`
  - 将 `minecraft { }` 配置块改为 `neoForge { }`
  - 将 `forge_version` 引用改为 `neo_version`
  - 更新 `processResources` 中的 `filesMatching` 从 `mods.toml` 改为 `neoforge.mods.toml`
  - 更新依赖从 `minecraft "net.minecraftforge:forge:..."` 改为 `neoForgeLibrary "net.neoforged:neoforge:..."`

## Task 4: 重命名和更新 mods.toml 为 neoforge.mods.toml

**Files:**
- Rename: `submodules/shanmod/src/main/resources/META-INF/mods.toml` → `submodules/shanmod/src/main/resources/META-INF/neoforge.mods.toml`

```toml
modLoader="javafml"
loaderVersion="${neo_version_range}"
license="${mod_license}"

[[mods]]
modId="${mod_id}"
version="${mod_version}"
displayName="${mod_name}"
description='''${mod_description}'''
authors="${mod_authors}"

[[dependencies.${mod_id}]]
    modId="neoforge"
    mandatory=true
    versionRange="${neo_version_range}"
    ordering="NONE"
    side="BOTH"

[[dependencies.${mod_id}]]
    modId="minecraft"
    mandatory=true
    versionRange="${minecraft_version_range}"
    ordering="NONE"
    side="BOTH"
```

- [ ] **Step 1: 重命名 mods.toml 为 neoforge.mods.toml**
  - 文件位于 `submodules/shanmod/src/main/resources/META-INF/`

- [ ] **Step 2: 更新 neoforge.mods.toml 内容**
  - 将 `loaderVersion="${loader_version_range}"` 改为 `loaderVersion="${neo_version_range}"`
  - 将 `modId="forge"` 依赖改为 `modId="neoforge"`
  - 将 `versionRange="${forge_version_range}"` 改为 `versionRange="${neo_version_range}"`
  - 更新 `mods.toml` 文件内容，移除无效的 license property

## 注意事项

1. **Java 21 是必需的** - NeoForge 1.21.1 需要 Java 21 才能运行
2. **需要安装 JDK 21** - 如果没有 ms-21 JDK，需要下载安装
3. **NeoForge 与 Forge API 兼容性** - 大部分 `net.minecraftforge` 包在 NeoForge 中仍然可用
4. **Parchment Mappings** - NeoForge 推荐使用 Parchment 来获得更好的参数名称
5. **版本号** - NeoForge 使用 `21.1.x` 格式对应 Minecraft `1.21.x`
